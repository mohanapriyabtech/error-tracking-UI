## Installation

Step 1: Clone the repository

   ```sh
   git clone https://github.com/mohanapriyabtech/error-tracking-UI.git
   ```

Step 2: Install dependencies

   ```sh
   cd error-tracking-UI
   npm install
   ```
 
Step 3: Start using the application in your browser

  ```sh
  npm start
  ```
##  React Version
